# 🍽️ Spice Garden – Restaurant Landing Page

A fully responsive restaurant landing page for an Indian restaurant, built with **HTML, CSS, and a small amount of JavaScript** for menu filtering.

---

## 📌 About This Project

I built this project to practice creating real-world landing pages with multiple sections. I wanted to design something that looked professional enough to be used by an actual small business. This helped me improve my CSS layout skills and JavaScript DOM manipulation.

---

## ✨ Features

- Fully **responsive** layout (mobile, tablet, desktop)
- Sticky dark navigation bar
- Full-screen hero section with overlay effects
- Highlights / info strip section
- **Filterable menu** — filter by Veg / Non-Veg / Dessert using JavaScript
- About section with restaurant stats
- Reservation form section
- Warm, food-appropriate color theme

---

## 🛠️ Built With

| Technology | Purpose |
|---|---|
| HTML5 | Semantic page structure |
| CSS3 | Styling, Grid, Flexbox, transitions |
| JavaScript (Vanilla) | Menu category filtering |
| Google Fonts | Playfair Display + Jost typography |

## 🚀 How to Run Locally

1. Clone the repo:
   ```bash
   git clone https://github.com/roshini15nivada/restaurant-landing-page.git
   ```
2. Open `index.html` directly in your browser. No server needed!
